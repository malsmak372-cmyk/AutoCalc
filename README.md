# AutoCalc

حاسبة أندرويد بسيطة للغاية.

هذه حاسبة أندرويد بسيطة للغاية، وهي مشروع تعليمي لتطوير تطبيقات الأندرويد.

### المميزات:
* حاسبة قياسية
* حاسبة علمية ودوال رياضية
* حسابات منطقية
* حساب فاصل التاريخ
* تحويل الوحدات
* تحويل المبالغ إلى كلمات (بالصينية حالياً)
* تحليل وحساب الصيغ التلقائي

تختلف هذه الحاسبة عن الحاسبات العادية في قدرتها على تحليل الصيغ التي تدخلها وحساب النتيجة مباشرة.

**ملاحظة:** هذا التطبيق قد يحتوي على بعض الأخطاء، يرجى عدم استخدامه في الحسابات الدقيقة للغاية.
